Conditions et termes

Bienvenue sur la plateforme d’enregistrement en ligne au concours d’entrée à l’Institut Ucac-Icam.
Pour l’enregistrement, vous aurez besoin :
  • De l’identité de l’élève (nom, prénom, date de naissance, établissement, série du baccalauréat clôturant le cycle secondaire…)
  • Du contact téléphonique et de l’identité des parents
  • D’un compte de paiement en ligne (orange money, Momo, Airtel)
  • Le payement des frais du concours se fait via le compte marchand de l'institut
Pour le CAMEROUN :
Pour l’enregistrement, vous aurez besoin :
  • Insérer le code #150*47# sur votre téléphone
  • Puis entrer le code : 381721
  • Puis le montant à payer (l’application d’inscription affichera le montant à payer selon le nombre de parcours sélectionnés).
  • Après paiement, votre opérateur vous enverra immédiatement un code d’identification de votre transaction débutant par les lettres MP suivi de plusieurs chiffres.
  • Insérer ce code dans le champ “’référence de paiement” sur la plateforme et insérer le numéro utiliser pour faire la transaction afin de finaliser votre inscription

  • Après avoir pris votre code d’identification dans votre centre d’examen, veuillez retourner dans l’application d’inscription et insérer ce code dans le champ “’référence de paiement” sur la plateforme et poursuivez votre enregistrement.
